Carteira

UMLUseCase
Untitled :: Model1 :: Carteira

Description

Fluxo Principal 1. O usuário pode indicar grupos determinados para as ações 2. As ações aparecem organizadas na carteira de acordo com os grupos delegados 3. O usuário pode solicitar a cotação das ações que tem em carteira 4. A solicitação é enviada para a corretora 5. A corretora solicita a cotação da ação pra B3 6. A B3 devolve o valor da cotação 7. A corretora devolve para o usuário o valor 8. O sistema tem um botão chamado "negocio rápido" em que o usuário tem um atalho para comprar ou vender uma ação que ele já tenha 9. O sistema mostra recomendações de compra e venda para o usuário com base no histórico dele 10. O usuário pode simular a compra e venda de uma ação que ele tem interesse
