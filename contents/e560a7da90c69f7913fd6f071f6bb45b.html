Acesso ao Sistema

UMLUseCase
Untitled :: Model1 :: Acesso ao Sistema

Description

Fluxo Principal 1. Utilizando o login e senha o usuário pode acessar o sistema (Fluxo Alternativo 01 e 02) 2. O usuário pode modificar suas informações cadastradas 3. O usuário pode habilitar/desabilitar alertas de cotação por email ou SMS 4. O sistema atualiza a data/hora do ultimo login feito

Fluxo Alternativo - 01 1.1. O usuário pode solicitar a recuperação da senha por email 1.2. O usuário acessa o link no email e cria uma nova senha 1.3. O usuário pode acessar o sistema com a nova senha

Fluxo Alternativo - 02 1.1. Após 5 tentativas de login falhas o usuário é bloqueado no sistema 1.2. O usuário recebe uma notificação por email ou SMS avisando que a conta foi bloqueada
